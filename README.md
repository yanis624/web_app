# \## 📋 Présentation

# 

# VERIDEM est une application web développée en une semaine lors du Marathon du Web 2026, un hackathon réunissant les étudiants de M1 MIASHS et M1 Info-Com (parcours CNO) de l'Université Paul-Valéry Montpellier 3.

# À l'heure où les images générées par intelligence artificielle circulent massivement en ligne et alimentent la désinformation, DIGIDEM s'attaque à la détection et l'analyse des fake news visuelles. Le projet se décline en deux volets.

# 

# Le premier volet s'articule comme l'analyse des tendances de la désinformation visuelle

# À partir d'un dataset d'images, nous identifions les tendances des fake news visuelles sur la période 2025-2026 : thématiques récurrentes, techniques de manipulation, formats les plus utilisés. Ces résultats sont mis en perspective grâce à une comparaison statistique avec les données 2024-2025, afin de dégager les évolutions et les nouvelles dynamiques de la désinformation par l'image.

# 

# Et le second volet implémente un détecteur d'images générées par IA

# Une interface web permet à l'utilisateur d'uploader une image et d'obtenir en retour :

# Une probabilité que l'image ait été générée par une IA

# Une identification des zones suspectes de l'image (quelles parties sont probablement générées)
